## Repository Interface Contracts

### EventRepository.getAll()
- **Returns**: `Event[]`
- **Notes**: Returns all events regardless of status. Filtering by status will be done on the service layer

### EventRepository.getById(id: number)
- **Returns**: `Event | null`
- **Notes**: Returns null if event isn't found. Never throws an error

### EventRepository.create(data: Omit<Event, "id" | "createdAt" | "updatedAt">)
- **Returns**: `Event`
- **Notes**: id, createdAt, and updatedAt are generated by the repository. status defaults to "draft" if not provided
